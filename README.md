# Confetti Chrome Extension

A Chrome extension that adds confetti to any webpage. Built as part of **Codecademy's AI Maker Bootcamp**.

## What It Does

Click the extension icon in your toolbar to open the popup, then press **"Pop confetti"** to trigger a confetti animation on the current tab. Works on any regular webpage (http/https).

## How to Use

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Build the extension**
   ```bash
   npm start
   ```

3. **Load in Chrome**
   - Go to `chrome://extensions/`
   - Turn on **Developer mode**
   - Click **Load unpacked**
   - Select the `build` folder

4. **Use it** — Click the extension icon, then "Pop confetti" on any webpage.

## Requirements

- Node.js 18 or higher
